# Local-only and air-gapped mode

Use this topic when Mery blocks remote providers, catalog refresh, or model downloads because local-only or air-gapped mode is active.

1. Use already installed local models and bundled catalogs.
2. Disable remote provider requests in the client while local-only mode is enabled.
3. Move verified artifacts into the local model store through approved install flows.

Air-gapped mode intentionally prevents network recovery actions.
