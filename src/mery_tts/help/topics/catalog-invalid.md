# Catalog invalid, signature, or checksum

Use this topic when the voice catalog is invalid, has a bad signature, or references artifacts with checksum mismatch.

1. Keep using the previous valid bundled or cached catalog when available.
2. Retry catalog refresh only when network policy allows it.
3. Do not install models from a catalog that failed validation.

Catalog validation failures are recoverable only from a trusted previous catalog or a fresh verified catalog.
